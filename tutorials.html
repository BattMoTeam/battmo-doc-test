Getting StartedÔÉÅ

Here, we present some guidelines and tutorials to get started with and give a short description of the example scripts that are available in BattMo

  • BattMo supports json inputs. The json input is specified in schemas, which also can be used to validate your input data. Add page for schema and link it here. Some standard data sets are included add page on that and link it here and add much more

    First, we load and parse the json input file using the commands

    filename = fullfile('ParameterData', 'BatteryCellParameters', 'LithiumIonBatteryCell', 'lithium_ion_battery_nmc_graphite.json')
jsonstruct = parseBattmoJson(filename)

    Then, we run it as follows

    runBatteryJson(filename)

    To visualize the results, we refer to the add link

  • In this json example, we show how to combine different json files

  • The battMoTutorial describes the different step of a simulation when it is set manually beyond the options that are given by the json input format. They consist of

    • Setup model parameters (physical parameters and modeling choice)

    • Setup the geometry (BattMo supports a variety of geometry)

    • Initialise the model

    • Setup a simulation schedule (time stepping and control)

    • Setup the initial state

    • Run the simulation

    • Visualize the results

  • In this example, more options are used such as some settings of the non-linear solver.
